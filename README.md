# Обработка метаданных изображений
Лабораторная работа номер 2.
Проект построен на Spring Boot и Spring Boot и WebFlux.

Цели: 
- создать и настроить WebClient и REST-сервисы(на  SpringBoot !);
- обрабатывать асинхронные и реактивные потоки данных (Mono/Flux);
- применять фильтры для логирования и обработки HTTP-запросов;
- работать с неоптимальной бизнес-логикой сервиса и готовиться к последующему профилированию и оптимизации. 

Задачи: 
- Создать Сервис А (клиент), который отправляет запросы на сервер и обрабатывает ответы через WebClient. 
- Создать Сервис B (сервер), который выполняет вычисления или обработку данных и возвращает результаты в реактивном формате (Mono/Flux). 
- Применять операторы фильтрации, трансформации и объединения данных. 
- Логирование всех входящих и исходящих запросов. 
- Обработка ошибок и исключений. 
- Настройка таймаутов и повторных попыток (retry backoff). 
- Реализовать операции с коллекциями, вычисления или поиск данных неэффективным способом (например, вложенные циклы, повторные сортировки, дублирующие преобразования).

Идея: 
A → запрос получение метаданных по ID. 
B → извлекает метаданные изображений. 
Задача: - Реализовать WebClient запрос. - Обработать Mono/Flux. - Покрыть логирование через фильтры. 
(Неоптимальная) логика: при запросе B загружает псевдо-изображение 
полностью в память, парсит EXIF вручную через рефлексию/библиотеки, 
повторно сериализует метаданные несколько раз. 
Что нужно сделать: реализовать загрузку изображения + парсинг с 
созданием тяжелых объектов. 

Данная программа сделана таким образом, чтобы выполнять задачу максмально неоптимальным образом. Картинка грузится в память полностью, выполняются ненужные и очень "жрущие операции", которые впустую прожигают озу и процессор.

Для работы проекта нужно иметь Java версии 17 или выше, а также другие необходимые утилиты.
Для запуска проекта необходимо скачать архив и затем, распаковав его, открыть две папки: service-a-client и service-b-client. В командной строке, находясь в указанных папках нужно запустить программы, командой .\mvnw spring-boot: run из обеих папок. Чтобы загрузить метаданные нужно перейти по ссылке http://localhost:8080/client/images/1/metadata.
